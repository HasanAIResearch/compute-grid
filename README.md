# The Compute Grid

Public GitHub Pages research page.

## Publish
Create a public GitHub repository named `compute-grid`, push this folder to `main`, then enable:

Settings -> Pages -> Deploy from branch -> main / root.

Expected free URL:
`https://HasanAIResearch.github.io/compute-grid/`

## Positioning

The public thesis:

AI is becoming a physical infrastructure race across compute, power, land, cooling, interconnection, silicon, and capital.

## Public-Safety Checklist

- No private portfolio, account, IBKR, or HH scanner details.
- Sources checked are listed on the page.
- Rounded/estimated values are described as rounded or illustrative.
- Page is research commentary, not personal investment advice.

## Local Verification

Open `index.html` directly in a browser before publishing. The page is self-contained and does not require a build step.
